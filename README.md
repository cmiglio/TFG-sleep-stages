# TFG-sleep-stages

Pasos para ejecutar el código:

1. Para empezar, hay que abrir la función **"config_function-example.m"** y copiarla y guardarla
con el nombre **"config_function.m"**. 
2. En la línea 6 de esta función hay que cambiar el "path" por el sitio donde estén guardadas las 
señales.
3. Finalmente, se puede ejecutar el script **"plot_rapido.m"**, para dibujar un segmento de 300 segundos 
de señal.
